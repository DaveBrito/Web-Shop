Problemas ao adicionar components no terminal:
Caso nao consiga pelo terminal do bash, 
usar o comando 'ng generate component home'. Entre no git bash, navegue ate a pasta do projeto, e por lá faça o comando para gerar o componente.
Ex: 'C:/Projeto/Angular > ng new component home'.

Problema com init git:
Caso já exista o git init no projeto, mas não está na pasta atual, remova-o com o comando.
 'rm -rf .git'
